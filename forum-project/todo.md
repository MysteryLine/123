## 🚀 立即开始

### 第一步：安装环境
按照 `SETUP.md` 中的说明安装 Node.js 和 MongoDB。

### 第二步：安装依赖
```bash
# 后端
cd backend && npm install

# 前端 (新终端)
cd frontend && npm install
```

### 第三步：启动服务
```bash
# 后端 (终端 1)
cd backend && npm run dev

# 前端 (终端 2)
cd frontend && npm run dev
```

### 第四步：打开浏览器
访问 http://localhost:3000

## 💡 开发建议

1. **从登录功能开始** - 这是最基础的功能
2. **逐步构建页面** - 先做列表，再做详情
3. **频繁测试** - 使用 Postman 或 Thunder Client 测试 API
4. **查看日志** - 检查浏览器控制台和后端日志
5. **遵循规范** - 参考 `copilot-instructions.md` 中的代码规范
